**Nala Vaariyum** is an AI-powered web portal developed to simplify access to **Tamil Nadu Labour Welfare Board schemes and services**. The platform enables workers to easily explore welfare schemes, understand eligibility criteria, and receive instant assistance through an integrated AI chatbot.

The application transforms complex government welfare information into a simple, interactive, and bilingual experience, making public services more accessible through modern web technologies.
